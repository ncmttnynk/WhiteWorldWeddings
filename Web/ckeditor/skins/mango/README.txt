Mango Skin for CKEditor, version 1.0

Mango was designed to be smoother, more beautiful replacement for skins packaged with CKEditor. It is based on "V2" theme, with icons taken from "Office2003" (from old FCKEditor).

Download

You can download Mango from CKSource forums:

http://cksource.com/forums/viewtopic.php?f=19&t=23667

Installation

1. Extract the contents of the file into the "skins" folder of CKEditor.
2. In the CKEditor configuration file (config.js) add the following:

[code]
config.skin = 'mango';
[/code]
